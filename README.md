# work-system-by-github
[基于 GitHub 的作业统计系统 ]( http://47.106.75.132 )

## 两个分支：
* develop: 程序代码
* master: 发布代码

## 功能
* 教师 
  1. 教师邮箱注册（邮箱发送验证码）、登录
  4. 查看个人信息
  1. 发布课程、任务
  2. 任务完成统计情况
  3. 批改学生作业、写评语
* 学生
  1. 学生关联GitHub
  3. 查看个人信息
  1. 查看课程及任务列表、领取任务
  2. 查看评语

## 执行
```
安装依赖
npm install

运行项目
npm run dev
```
